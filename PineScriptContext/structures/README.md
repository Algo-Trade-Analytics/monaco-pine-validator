# Enhanced Pine Script Structures

This directory contains TypeScript structures automatically generated from scraped Pine Script v6 documentation.

## Generation

These files are generated by `scripts/generate-enhanced-structures.ts` from `PineScriptContext/pinescript_reference.jsonl`.

## Structure

Each file contains rich metadata including:
- **Variables**: name, qualifier, type, description, remarks, examples, see also references
- **Functions**: name, description, syntax, parameters, returns, examples, remarks, see also references
- **Constants**: name, qualifier, type, description, remarks, see also references
- **Keywords**: name, description, syntax, examples, remarks, returns, see also references
- **Operators**: name, description, syntax, examples, remarks, returns
- **Types**: name, description, fields, examples, remarks, see also references
- **Annotations**: name, description, examples, remarks

## Usage

```typescript
import { pineScriptDocumentation } from './PineScriptContext/enhanced-structures';

// Access variable metadata
const askVar = pineScriptDocumentation.variables.ask;
console.log(askVar.description); // "The ask price at the time..."

// Access function metadata
const alertFunc = pineScriptDocumentation.functions.alert;
console.log(alertFunc.syntax); // "alert(message, freq) → void"
console.log(alertFunc.parameters); // [{ text: "message (series string)..." }]

// Access constant metadata
const colorRed = pineScriptDocumentation.constants.color.red;
console.log(colorRed.description); // "Is a named constant for #F23645 color."
```

## Advantages over Manual Structures

1. **Complete metadata**: Includes descriptions, examples, remarks, and cross-references
2. **Always up-to-date**: Regenerated from official documentation
3. **Rich type information**: Full syntax and parameter information for functions
4. **Better IDE support**: Comprehensive JSDoc-style information
5. **Validation**: Can be used for enhanced error messages and autocomplete

## Files

- `variables.ts` - All Pine Script variables (160 entries)
- `functions.ts` - All Pine Script functions (457 entries)
- `constants.ts` - All Pine Script constants (238 entries)
- `keywords.ts` - Pine Script keywords (15 entries)
- `operators.ts` - Pine Script operators (21 entries)
- `types.ts` - Pine Script types (18 entries)
- `annotations.ts` - Pine Script annotations (10 entries)
- `index.ts` - Main export file

## Regeneration

To regenerate these structures:

```bash
npx tsx scripts/generate-enhanced-structures.ts
```
